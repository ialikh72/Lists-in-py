Movie List Collector
A simple Python script that prompts the user for their favorite movies and stores them in a dynamic list.

📝 Description
This script demonstrates basic Python input handling and list manipulation. It captures three movie names from the user, appends them to a list called movies, and then prints the final collection to the console.

🚀 How It Works
Initialization: An empty list named movies is created.

Input: The script uses the input() function three times to gather data from the user.

Storage: Each entry is added to the list using the .append() method.

Output: The complete list is displayed in the terminal.
